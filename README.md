# 2026 Challenge Taiwan 113 訓練計劃

16 週鐵人三項訓練計劃網站，為 2026 年 Challenge Taiwan 113 半程超級鐵人三項賽事設計。

## 賽事資訊

- **賽事名稱**: Challenge Taiwan 113
- **日期**: 2026年4月25日
- **地點**: 台東
- **距離**:
  - 游泳 1.9km
  - 自行車 90km
  - 跑步 21.1km

### 運動顏色對應

- 游泳: #63D9ED (Cyan)
- 自行車: #FF9405 (Orange)
- 跑步: #05F06B (Green)

## 功能特色

- 16週完整訓練計劃
- 比賽倒數計時器
- 每日訓練內容顯示
- 訓練週期視覺化圖表
- Garmin Connect 整合
- 響應式設計支援手機瀏覽

## 技術架構

- HTML5 / CSS3 / JavaScript
- Chart.js 資料視覺化
- Vercel Serverless Functions (Garmin API)

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npx serve .
```

## 部署

本專案支援 Vercel 部署，已包含 `vercel.json` 配置檔案。
